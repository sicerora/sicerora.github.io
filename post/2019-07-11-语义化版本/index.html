语义化版本

文档索引 | 文档链接：https://semver.org/lang/zh-CN/
2019年7月11日
| 代码管理

版本格式：主版本号.次版本号.修订号，版本号递增规则如下：
主版本号：当你做了不兼容的 API 修改，
次版本号：当你做了向下兼容的功能性新增，
修订号：当你做了向下兼容的问题修正。
